# Email Notification Setup Guide

## Goal

Send registration data to `shivampan707@gmail.com` automatically when someone submits the form.

---

## Method 1: Using Google Apps Script (RECOMMENDED - Free & Automatic)

### Step 1: Go to Your Google Form

1. Open: https://docs.google.com/forms/d/e/1FAIpQLSeOz70Vr-blHr4ueCsVGCRsxuo30YsyDU7P-lEbPdTSDlweTA/edit
2. Click the **three-dot menu** (⋮) in the top right
3. Select **"Script editor"**

### Step 2: Copy & Paste the Script

Delete any existing code and paste this:

```javascript
// Configuration
const RECIPIENT_EMAIL = "shivampan707@gmail.com";
const SHEET_NAME = "Form Responses 1"; // Default Google Forms sheet name

// Function triggered when form is submitted
function onFormSubmit(e) {
  try {
    // Get the response
    const response = e.response;
    const itemResponses = response.getItemResponses();

    // Create email body
    let emailBody = "📝 New Registration Submitted!\n\n";
    emailBody += "=" + "=".repeat(50) + "\n";
    emailBody += "REGISTRATION DETAILS\n";
    emailBody += "=" + "=".repeat(50) + "\n\n";

    let registrationData = {};

    for (let i = 0; i < itemResponses.length; i++) {
      const itemResponse = itemResponses[i];
      const title = itemResponse.getItem().getTitle();
      const response_value = itemResponse.getResponse();

      emailBody += `✓ ${title}: ${response_value}\n`;
      registrationData[title] = response_value;
    }

    emailBody += "\n" + "=" + "=".repeat(50) + "\n";
    emailBody += `📅 Submitted at: ${new Date().toLocaleString()}\n`;
    emailBody += "=" + "=".repeat(50) + "\n\n";

    emailBody += "📊 Responses tracked in Google Sheet:\n";
    emailBody +=
      "https://docs.google.com/spreadsheets/d/YOUR_SHEET_ID/edit\n\n";

    emailBody += "---\n";
    emailBody += "⚡ Auto-generated by Drone Camp Registration System\n";

    // Send email
    GmailApp.sendEmail(
      RECIPIENT_EMAIL,
      "✈️ New Drone Camp Registration - " + registrationData["👤 Full Name"],
      emailBody,
    );

    console.log("✅ Email sent to " + RECIPIENT_EMAIL);
  } catch (error) {
    console.error("❌ Error sending email: " + error);

    // Send error notification
    GmailApp.sendEmail(
      RECIPIENT_EMAIL,
      "⚠️ Error in Drone Camp Registration Email",
      "Error: " + error.toString(),
    );
  }
}
```

### Step 3: Set Up the Trigger

1. In the Script Editor, click the **"Clock Icon"** (Triggers) on the left
2. Click **"+ Create new trigger"**
3. Set up like this:
   - **Choose which function to run:** `onFormSubmit`
   - **Which runs at deployment:** `Head`
   - **Select event type:** `On form submit`
   - Click **"Create"**
4. Grant permissions when prompted

### Step 4: Deploy the Script

1. Click **"Deploy"** (top right)
2. Click **"New deployment"**
3. Select **"Type"** → **"Web app"**
4. Set:
   - Execute as: Your account
   - Who has access: Anyone
5. Click **"Deploy"**
6. Click **"Allow"** when prompted

✅ **Done!** Now you'll receive emails for every registration.

## Method 2: Add Direct Email to Your Website (Alternative)

If Method 1 doesn't work, modify your `script.js`:

```javascript
// Add this to your submitRegistration function
fetch("https://api.emailjs.com/api/v1.0/email/send", {
  method: "POST",
  headers: {
    "phase/json": "application/json",
  },
  body: JSON.stringify({
    service_id: "your_service_id",
    template_id: "your_template_id",
    user_id: "your_emailjs_id",
    template_params: {
      to_email: "shivampan707@gmail.com",
      name: nameValue,
      email: emailValue,
      phone: phoneValue,
      age: ageValue,
      guardian: guardianValue,
      batch: batchValue,
      school: schoolValue,
    },
  }),
});
```

This requires setting up EmailJS at: https://www.emailjs.com/

## Troubleshooting

| Problem               | Solution                                            |
| --------------------- | --------------------------------------------------- |

| Script not triggering | Ensure trigger is set to "On form submit"           |
| Permission denied     | Grant necessary permissions when prompted           |
| Wrong sheet name      | Change "Form Responses 1" to your actual sheet name |
